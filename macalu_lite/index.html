🔧 Edge Computing

Lokale Autonomie-Engine für Echtzeit-Entscheidungen im Fahrzeug. Führt grundlegende Fahrmanöver durch und bereitet Daten für das Gateway vor. Kritische Entscheidungen wie Notbremsung erfolgen in unter einer Millisekunde.

  • Lokale autonome Entscheidungen
  • Hardware-optimierte ML-Modelle
  • TPM-basierte Sicherheit

Zentrale Autonomie-Engine in der Cloud für globale Optimierung, Risikobewertung und ethische Entscheidungen. Verarbeitet Telemetriedaten von Millionen von Fahrzeugen für übergeordnete Strategien.

  • Globale Routenoptimierung
  • Risikobewertung und Prognosen
  • Ethische Entscheidungsmodelle
  • Streaming-Analytics

Security Policy Agent

Vehicle Event Gateway

  • Client-Zertifikat-Authentifizierung
  • Telemetrie-Veröffentlichung
  • Cloud-Befehlsempfang
